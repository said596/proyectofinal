Estructuras de Bifurcación

Introducción

Las estructuras de bifurcación, también conocidas como estructuras de control de flujo, son fundamentales en la programación y permiten que un programa tome decisiones y ejecute diferentes bloques de código en función de condiciones específicas. En C, las estructuras de bifurcación más comunes son if, if-else, else if, y switch.

Estructuras de Bifurcación en C

1. if

La estructura if se utiliza para ejecutar un bloque de código solo si una condición específica se evalúa como verdadera.

#include <stdio.h> int main() { int x = 10; if (x > 5) { printf("x es mayor que 5\\n"); } return 0; }

2. if-else

La estructura if-else se utiliza para ejecutar un bloque de código si una condición es verdadera y otro bloque si la condición es falsa.

#include <stdio.h> int main() { int x = 10; if (x > 15) { printf("x es mayor que 15\\n"); } else { printf("x no es mayor que 15\\n"); } return 0; }

3. else if

La estructura else if permite comprobar múltiples condiciones y ejecutar diferentes bloques de código según cuál condición se evalúe como verdadera.

#include <stdio.h> int main() { int x = 10; if (x > 15) { printf("x es mayor que 15\\n"); } else if (x > 5) { printf("x es mayor que 5 pero menor o igual a 15\\n"); } else { printf("x es menor o igual a 5\\n"); } return 0; }

4. switch

La estructura switch se utiliza para seleccionar entre múltiples bloques de código basados en el valor de una expresión.

#include <stdio.h> int main() { int x = 2; switch (x) { case 1: printf("x es 1\\n"); break; case 2: printf("x es 2\\n"); break; case 3: printf("x es 3\\n"); break; default: printf("x no es 1, 2 o 3\\n"); break; } return 0; }
