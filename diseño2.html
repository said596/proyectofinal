Diseño Centrado en el Género

1. Comprensión del Contexto de Género

Es fundamental entender cómo el género influye en la experiencia y necesidades de los usuarios. Esto implica investigar las desigualdades de género existentes y considerar cómo estas pueden afectar el diseño del producto o servicio.

2. Investigación y Recopilación de Datos

Recolectar datos específicos relacionados con el género a través de encuestas, entrevistas, grupos focales y otras técnicas de investigación. Esto ayuda a identificar las necesidades y preferencias de diferentes grupos de género.

3. Definición de Objetivos y Requisitos

Establecer objetivos claros y requisitos específicos para el diseño que aborden las necesidades de género identificadas durante la investigación.

4. Diseño Inclusivo

Desarrollar soluciones de diseño que sean inclusivas y equitativas, asegurando que todos los usuarios, independientemente de su género, puedan beneficiarse del producto o servicio.

5. Evaluación y Validación

Probar y evaluar el diseño con usuarios reales para asegurar que cumple con los objetivos de inclusión y equidad de género. Recoger feedback y hacer ajustes necesarios.

6. Implementación y Monitoreo

Implementar el diseño final y monitorear su uso y efectividad en términos de equidad de género. Hacer mejoras continuas basadas en el feedback y la observación.

Ejemplo de Código en C

#include <stdio.h>
#include <string.h>

#define MAX 100

typedef struct {
    char gender[10];
    char preference[50];
} User;

void collectData(User users[], int *count) {
    char cont = 'y';
    while(cont == 'y' && *count < MAX) {
        printf("Ingrese su género (masculino, femenino, otro): ");
        scanf("%s", users[*count].gender);
        printf("Ingrese su preferencia de diseño: ");
        scanf("%s", users[*count].preference);
        (*count)++;
        printf("¿Desea ingresar otro registro? (y/n): ");
        scanf(" %c", &cont);
    }
}

void displayData(User users[], int count) {
for(int i = 0; i < count; i++) {
    printf("Género: %s, Preferencia: %s\n", users[i].gender, users[i].preference);
}
}

int main() {
    User users[MAX];
    int count = 0;

    collectData(users, &count);
    displayData(users, count);

    return 0;
}
