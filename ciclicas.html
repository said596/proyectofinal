Estructuras Cíclicas

Introducción

Las estructuras cíclicas, también conocidas como bucles o loops, son fundamentales en la programación y permiten que un conjunto de instrucciones se repita múltiples veces hasta que se cumpla una condición específica. En C, los bucles más comunes son for, while, y do-while.

Estructuras Cíclicas en C

1. for

La estructura for se utiliza para ejecutar un bloque de código un número específico de veces.

#include <stdio.h> int main() { int i; for (i = 0; i < 5; i++) { printf("Iteración %d\\n", i+1); } return 0; }

2. while

La estructura while se utiliza para ejecutar un bloque de código mientras una condición sea verdadera.

#include <stdio.h> int main() { int i = 0; while (i < 5) { printf("Iteración %d\\n", i+1); i++; } return 0; }

3. do-while

La estructura do-while se utiliza para ejecutar un bloque de código al menos una vez y luego repetirlo mientras una condición sea verdadera.

#include <stdio.h> int main() { int i = 0; do { printf("Iteración %d\\n", i+1); i++; } while (i < 5); return 0; }
